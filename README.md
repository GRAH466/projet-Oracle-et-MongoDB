# Sauvegarde et restauration des bases de données & manipulation MongoDB avec Python

## Auteur

Grah Lowa Vincent Ezechiel

## Description du projet

Ce dépôt regroupe deux travaux réalisés dans le cadre de ma formation en informatique :

1. **Un rapport de groupe sur la sauvegarde et la restauration des bases de données avec Oracle.**
2. **Un projet pratique individuel consistant à créer, enregistrer et manipuler une base de données MongoDB à l’aide de Python.**

Ces travaux ont pour objectif de comprendre les techniques de gestion, de sauvegarde et de manipulation des bases de données dans différents environnements.

## Partie 1 : Sauvegarde et restauration avec Oracle

Cette partie présente un rapport expliquant les concepts et les méthodes de sauvegarde et de restauration des bases de données dans Oracle.

### Objectifs

* Comprendre l'importance de la sauvegarde des bases de données
* Étudier les différentes méthodes de sauvegarde dans Oracle
* Apprendre les techniques de restauration en cas de perte ou de corruption des données
* Garantir la sécurité et la disponibilité des données

### Contenu

* Présentation du système Oracle
* Types de sauvegardes
* Processus de restauration
* Bonnes pratiques de gestion des bases de données

## Partie 2 : Manipulation d'une base de données MongoDB avec Python

Cette partie consiste à créer et manipuler une base de données **MongoDB** à l’aide du langage **Python**.

### Objectifs

* Créer une base de données MongoDB
* Insérer des données dans les collections
* Lire et afficher les données
* Modifier et supprimer des documents
* Comprendre l’utilisation de MongoDB avec Python

### Technologies utilisées

* Python
* MongoDB
* Visual Studio Code

### Fonctionnalités réalisées

* Connexion à la base de données MongoDB
* Création d'une base de données et de collections
* Insertion de documents
* Lecture des données
* Mise à jour des documents
* Suppression des données

## Objectif pédagogique

Ce projet permet de comparer les approches de gestion de bases de données entre :

* les bases relationnelles (Oracle)
* les bases NoSQL (MongoDB)

Il permet également de développer des compétences en gestion, sauvegarde et manipulation des données.
